## Session 3 Updates

### Improved Folder Discovery

The find-folder endpoint now:
- Searches deeper in user directories (up to 6 levels in Pictures/Videos)
- Checks more drive letters (D: through H:)
- Uses per-path depth limits for faster searching
- Skips more system folders (Windows, Program Files, etc.)

### Manual Path Input (Cross-Device Fallback)

When the server can't automatically locate a folder:
1. A modal prompts for the full path (e.g., `D:\Photos\SFW`)
2. User can paste the path from File Explorer
3. Server-side scan enables cross-device access
4. "Skip (Local Only)" option for device-local access only
